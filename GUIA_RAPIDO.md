# Guia Rápido de Uso - OAIBV

## Iniciando o Sistema

### Windows
1. Dê um duplo clique no arquivo `iniciar_oaibv.bat`
2. O sistema será iniciado e o navegador abrirá automaticamente

### Linux/Mac
1. Dê um duplo clique no arquivo `iniciar_oaibv.sh` ou execute-o no terminal com `./iniciar_oaibv.sh`
2. O sistema será iniciado e o navegador abrirá automaticamente

## Acessando o Sistema

- **Usuário Administrador**: admin
- **Senha Padrão**: admin123

## Funcionalidades Principais

### Inventário
- Visualização completa de todos os itens cadastrados
- Filtros por grupo e busca por tombamento/descrição
- Opção de edição para usuários administradores

### Transações
- Registro de entradas e saídas de itens
- Controle de quantidade disponível

### Empréstimos
- Registro de empréstimos de itens
- Geração de termo de compromisso
- Controle de devolução

### Relatórios
- Geração de relatórios de inventário, transações e empréstimos
- Exportação em PDF, CSV e Word
- Filtros por data, tipo e grupo

### Logs
- Registro de todas as ações realizadas no sistema
- Identificação do usuário responsável
- Data e hora da ação

## Dicas de Uso

- Mantenha o inventário atualizado para evitar inconsistências
- Utilize os filtros para facilitar a busca de informações
- Gere relatórios periódicos para acompanhamento
- Verifique os logs para auditar ações realizadas no sistema

## Suporte

Em caso de dúvidas ou problemas, entre em contato com o administrador do sistema.
